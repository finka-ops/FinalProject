# SysAdminTamagotchi
Sysadmin Tamagotchi is a terminal-based Tamagotchi pet game where the user keeps a “server pet” running for a set number of turns. The player chooses a pet type and then selects actions each turn (like feeding coffee, applying patches, restarting services, or ignoring alerts). Actions and random events change the pet’s stats (health, energy, mood).  The goal of the game is to keep the pet’s health above 0 through all turns to win.  If health hits 0 , the player loses and the server pet takes a nap.
